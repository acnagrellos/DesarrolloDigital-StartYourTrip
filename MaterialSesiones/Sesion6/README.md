5. ¿Qué se imprimirá al ejecutar el siguiente código?

```

if (a > b)
    Console.WriteLine("A es mayor");
else
    Console.WriteLine("B es mayor");
```

a) A es mayor
b) B es mayor (Respuesta correcta)
c) Error de compilación
d) Nada, el programa no imprime salida.
